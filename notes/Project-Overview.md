# South Indian Food Ordering System

## Project Goal

Build a production-ready food ordering application for:

- Android
- iOS
- Web

using

- React Native
- NestJS
- PostgreSQL
- Redis
- Docker
- Prisma

## Main Features

- Authentication
- Browse Food Items
- Categories
- Cart
- Checkout
- Static Paytm QR
- Order History
- Admin Panel
